💡 Quick Instructions: How to Use the Cart View

  • 🟢 Green Slots (Standard): Displays permanent roster seat assignment.
  • 🔄 Dropdown Selectors: Use the status selector on any slot card to change statuses instantly.
  • ⏱️ Automated Tracking: Status modifications log instantly to the transaction archives.

💡 Quick Instructions: Managing 1:1 Student Devices

  • 🔍 Global Filtering: Type any student name, grade, or asset barcode into the text search tool to filter.
  • 🔄 Asset Swapping: Click the device tag button to permanently issue a new asset number and archive the old one into history.
  • ⏱️ Loaner Allocation: Use status changes to provision loaner tags without modifying their primary serial number.
